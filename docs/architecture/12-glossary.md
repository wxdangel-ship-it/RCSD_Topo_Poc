# 12 术语表

- **Source-of-Truth**：当前被认定为正式事实来源的文档集合。
- **Durable Guidance**：长期稳定、但不承载业务真相的执行规则。
- **Skill**：可跨轮次复用的标准工作流说明。
- **TEXT_QC_BUNDLE**：用于内外网之间回传质检信息的结构化文本协议。
- **Patch**：当前初始化阶段沿用的最小数据处理单位目录组织。
- **模块启动模板**：`modules/_template/` 中提供的新模块初始文档契约骨架。
- **Support Retained**：已纳入仓库治理、但不属于当前业务生产闭环的支撑 / 工具集合模块。
