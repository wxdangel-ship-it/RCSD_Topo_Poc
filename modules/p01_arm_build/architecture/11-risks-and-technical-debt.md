# 11 风险与技术债

## 当前风险

- 真实数据字段名可能存在大小写或命名漂移。
- 右转专用道字段值尚未在项目级统一冻结；当前实现只能对 `--right-turn-formway-value` 显式传入的已确认值保守排除。
- T 型主通道 / 侧向判断在缺少可靠方向与配对证据时会偏保守，输出 ambiguous 而不是强行 through。
- 真实数据规模与字段完整性尚未在本轮验证。

## 缓解方式

- 字段读取大小写不敏感。
- 字段缺失写 issue，不反推语义。
- 所有停止点输出 trace 与 decision，供后续 case 归纳。
- 后续如需正式 CLI，走入口登记和契约同步。
