# PPT 页面目录

本文件是 PPT 母版的页面工作台。每页先记录结论和证据，再进入具体撰写。

## 字段说明

| 字段 | 说明 |
|---|---|
| 页码 | 母版页码，后续可调整。 |
| 标题 | PPT 页面标题。 |
| 章节 | 所属大章。 |
| 本页结论 | 听众看完这一页应接受的核心判断。 |
| 支撑证据 | 数据、文档、run root、审计产物或人工确认来源。 |
| 图片需求 | 是否需要流程图、漏斗图、case 图或截图。 |
| 精简版策略 | 面向领导精简版保留、合并或删除。 |
| 技术展开点 | 面向技术同事可展开的细节。 |
