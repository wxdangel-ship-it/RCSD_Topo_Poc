# 第二部分 T01 / T06 Segment 替换业务单元页组归档说明

## 归档状态

本组页面作为第二部分 T01 / T06 模块阶段性草稿归档，用于承接前序路口锚定业务单元，进入 Segment 构建、替换前提判断、RCSD Segment 构建、Replacement Plan 执行和路段替换质量定位。

## 页面定位

本组页面用于说明：

1. T01 如何基于 SWSD 构建稳定 Segment，定义后续替换与通行恢复的共同承载单元。
2. T06 如何消费 T01 Segment、T05 Relation 与 RCSD Road / Node，判断是否具备替换前提。
3. T06 如何构建 RCSD Segment 并证明是否可以安全替换。
4. T06 Step3 如何以 Replacement Plan 驱动替换执行，输出可审计 F-RCSD。
5. 路段替换质量问题如何从 Segment 边界、Relation 消费、RCSD 承载和执行闭合等多个质量门定位。
