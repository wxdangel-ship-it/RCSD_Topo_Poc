# 12 术语表

- `T00`：`RCSD_Topo_Poc` 中的 Utility Toolbox / 工具集合模块
- `Tool1`：Patch 数据整理脚本，负责建立统一 `patch_all` 骨架
- `Tool2`：全量 `DriveZone` 预处理与合并工具
- `Tool3`：全量 `Intersection` 预处理与汇总工具
- `PatchID`：Patch 目录名，对应 `patch_all/<PatchID>/`
- `patch_all`：Tool1 产出的统一 Patch 目录根
- `patchid`：Tool3 写入全局 `Intersection` 输出的来源 Patch 标识字段
